React生命周期详解

一个React组件的生命周期通常分为创建(初始化和挂载)、更新、销毁三个阶段，对于生命周期的每个阶段，React提供了不同的钩子函数，让我们去执行不同的操作。

组件初始化阶段(initialization)

  1. getDefaultProps()
  2. getInitialState()
  3. constructor()

在组件的初始化阶段，React提供的生命周期函数主要包含以上三种

但要特别注意他们之间的区别,getDefaultProps()方法和getInitialState()方法是当采用ES5的React.CreateClass方式创建组件时的内置生命周期函数,如果你采用ES6的extends React.Component方式创建组件，只有constructor()方法

即：ES5 getDefaultProps()、getInitialState（） = ES6 constructor()

由于ES5创建组件的方式已不在常用，因此这里主要介绍constructor()方法:

constructor(props)如果存在，要调用super(props),一般用来初始化state和绑定this，只执行一次

import React, { Component } from 'react';

class App extends Component {
  constructor(props) {
    super(props);
    this.state={
        age:'23'
    }
    this.handleClick = this.handleClick.bind(this)
  }
  handleClick(){
      console.log('handle的this指向：',this)
  }
  render(){
      return(
          <div>
            <button onClick={this.handleClick}>点击事件</button>
          </div>
      )
  }
}
